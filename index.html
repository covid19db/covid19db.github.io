Covid-19 DB Project


OBJECTIVE:
In response to the current coronavirus outbreak, we are creating a comprehensive, publicly available database of the severe acute respiratory syndrome coronavirus 2 (SARS-CoV-2) that causes coronavirus disease (COVID-19). We hope this would be useful resource to create detailed visualisation tools, build a model of the global spread of the virus and potentially inform adequate public policy.

The project is coordinated by researchers from the University of Oxford, however contributions from around the world are welcome and much appreciated. If you would like to help, please get in touch: adam.mahdi@eng.ox.ac.uk


HOW CAN YOU HELP?

  • If you are a developer, you can help us with developoing "fetchers" that collect data from machine-readable official sources and feed them into the database.
  • If you work for a health authority or statistics office, and able to contribute to this public resource please get in touch.

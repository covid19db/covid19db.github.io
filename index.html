Covid-19 DB Project


OBJECTIVE:

In response to the current coronavirus outbreak, we are creating a comprehensive, publicly available database of the severe acute respiratory syndrome coronavirus 2 (SARS-CoV-2) that causes coronavirus disease (COVID-19). He hope this would be useful to create detailed visualisation tools, build a model of the global spread of the virus and potentially inform adequate public policy strategy.


HOW CAN YOU HELP?

The project is coordinated by researchers from the Universit of Oxford, however contribution from around the world are welcome and much appreciated. If you want to help, please get in touch: adam.mahdi@eng.ox.ac.uk

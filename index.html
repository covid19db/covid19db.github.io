OXFORD COVID-19 (OxCOVID19) PROJECT

OxCOVID19 Project aims to increase our understanding of the Covid-19 pandemic and elaborate possible strategies to reduce the impact on the society through the combined power of Statistical, Mathematical Modelling and Machine Learning techniques.

OxCOVID19 Database is a large, single-centre, multimodal database consisting of information (using acknowledged data sources) related to COVID-19 pandemic. OxCOVID19 Database is currenlty comprised of six tables with a common key, combining different types of data and making it easy to query.

Accessing OxCOVID19 Database

You can access OxCOVID19 database in two ways. Connecting to the public PostgreSQL database, e.g. using the following code 

    import psycopg2
    import pandas as pd

    conn = psycopg2.connect(
    host='covid19db.org',
    port=5432,
    dbname='covid19',
    user='covid19',
    password='covid19'
    )
    cur = conn.cursor()

    # load EPIDEMIOLOGY table
    cur.execute("SELECT * FROM epidemiology LIMIT 100;")
    df_epidemiology = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load MOBILITY table
    cur.execute("SELECT * FROM mobility LIMIT 100;")
    df_mobility = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load GOVERNMENT_REPONSE table
    cur.execute("SELECT * FROM government_response LIMIT 100;")
    df_govres = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load COUNTRY STATISTICS table
    cur.execute("SELECT * FROM country_statistics LIMIT 100;")
    df_govres = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load WEATHER table
    cur.execute("SELECT * FROM weather LIMIT 100;")
    df_govres = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load ADMINISTRATIVE_DIVISION table
    cur.execute("SELECT * FROM administrative_division LIMIT 100;")
    df_admin = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

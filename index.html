Covid-19 DB Project


OBJECTIVE:
In response to the current coronavirus outbreak, we are creating a comprehensive, publicly available database of the severe acute respiratory syndrome coronavirus 2 (SARS-CoV-2) that causes coronavirus disease (COVID-19). He hope this would be useful to create detailed visualisation tools, build a model of the global spread of the virus and potentially inform adequate public policy.

WHAT DATA DO WE HAVE?

So far, have collected the data from the following sources:

Country Source Source code Granularity Statistics
World Data collected by Johns Hopkins from WHO reports WRD-JHWHO Mostly country-wise, sometimes includes states, provinces or oversea territories infected, recovered, dead
Italy Data collected by Davide Mango from Protezione Civile ITA-DMPC Italian regions tested, infected, hospitalised, in ICU, quarantined, recovered, dead

HOW CAN I USE THE DATABASE?

Currently, the schema includes only one table, infections, with the following columns:

  • source: code describing the source of information. See table above.
  • date: day of the statistics.
  • country: English name for the country.
  • countrycode: ISO ALpha-3 code of the country.
  • l1region: (optional) wide-area administrative or geographical region within the country, for example a "state" within United States, a "province" within China or Canada, or an oversea territory like Faroe Islands for Denmark or Martinique for France.
  • l2region: (optional) mid-area region within a country or a l1region, for example an Italian "region", a UK "county"
  • l3region: (optional) a small-area region within the country or the l2region, for example an italian "province".
  • city: (optional) a city, town or village.
  • tested: (optional) number of people tested. Cumulative.
  • infected: (optional) number of people who has been confirmed as infected. Cumulative.
  • recovered: (optional) number of people who has been infected and has healed. Cumulative.
  • hospitalised: (optional) number of people who has been infected and has been hospitalised. Cumulative.
  • hospitalised_icu: (optional) number of people who has been infected and has been hospitalised and had to be put in intensive care. Cumulative.
  • homequarantined: (optional) number of people who has been infected and has been quarantined in home. Cumulative.

We collect the data as they come from the data sources, we do not check for their consinsentcy nor correctness. It is possible that different sources provide slighlty different statistics.

Warning: the dattabase is being built and it is likely that the datastructure will change, check this webpage regularly to be notified about changes.

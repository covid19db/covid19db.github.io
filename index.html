Understanding global impact of Covid-19 through Data Science

OBJECTIVE

This project aims to increase our understanding of the COVID-19 pandemic, elaborate possible ways to slow down the spread of the virus and suggest strategies to reduce the impact on the society through the combined power of Statistical and Mathematical Modelling, and Machine Learning techniques.

We are currently in a process of assembling a comprehensive and conveniently formatted database COVID19DB (using acknowledged source), which is being made available to the public as we go. We aim to combine the following data:
  • Epidemiological data (e.g. confirmed, dead, etc.)
  • Government tracking (e.g. shool closures, quarantines, etc.)
  • Movement data (e.g. movement of people, travel, social distansing, etc.)

HOW CAN I CONNECT TO THE DATABASE?

You can use, for example, the following Python code and save the database into a DataFrame (df): 
    import psycopg2
    import pandas as pd

    conn = psycopg2.connect(
    host='covid19db.org',
    port=5432,
    dbname='covid19',
    user='covid19',
    password='covid19'
    )
    cur = conn.cursor()
    cur.execute("SELECT * FROM infections;")
    df = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

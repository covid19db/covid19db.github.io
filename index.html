COVID19DB PROJECT

This project aims to increase our understanding of the COVID-19 pandemic and elaborate possible strategies to reduce the impact on the society through the combined power of Statistical and Mathematical Modelling, and Machine Learning techniques.

We are currently in a process of assembling a comprehensive and conveniently formatted database COVID19DB (using acknowledged sources), which is being made available to the public as we go. We aim to combine the following data:
  • Epidemiology (e.g. confirmed, dead, recovered, hospitalised, etc.)
  • Mobility (movement trends over time by geography, e.g. workplace, parks)
  • Government response (e.g. shool closures, quarantines, etc.)
  • Country statistics (e.g. population, GDP, number of hospitas, etc.)

Accessing COVID19DB database

You can access COVID19DB database in two ways. Connecting to the public PostgreSQL database, e.g. using the following code 

    import psycopg2
    import pandas as pd

    conn = psycopg2.connect(
    host='covid19db.org',
    port=5432,
    dbname='covid19',
    user='covid19',
    password='covid19'
    )
    cur = conn.cursor()

    # load EPIDEMIOLOGY table
    cur.execute("SELECT * FROM epidemiology LIMIT 100;")
    df_epidemiology = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load MOBILITY table
    cur.execute("SELECT * FROM mobility LIMIT 100;")
    df_mobility = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

    # load GOVERNMENT_REPONSE table
    cur.execute("SELECT * FROM government_response LIMIT 100;")
    df_govres = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

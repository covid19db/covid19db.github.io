INFECTIONS table
Name Data type Short Description
SOURCE VARCHAR(40) Data source
DATE INT Day of the statistics
COUNTRY VARCHAR(40) English name for the country
COUNTRYCODE VARCHAR(10) ISO alpha-3 code of the country
ADM_AREA_1 VARCHAR(40) First-level administrative country subdivision
ADM_AREA_2 VARCHAR(40) Second-level administrative country subdivision
ADM_AREA_3 VARCHAR(40) Third-level administrative country subdivision
TESTED INT Number of people tested
CONFIRMED INT Third-level administrative country subdivision
RECOVERED INT Number of confirmed and infected who have recovered
HOSPITALISED INT Number of confirmed who are/have been hospitalised
HOSPITALISED_ICU INT Number of confirmed who are/have been in the intensive care
QUARANTINED INT Number of confirmed with home quarantine

Note 1: We collect the data as they come from the indicated data sources, we do not check for their consinsentcy nor correctness. It is possible that different sources provide slighlty different statistics.

Note 2: The database is being built and it is likely that the datastructure will change. Check this webpage regularly to be notified about changes.


DATA SOURCES

So far, we have used the following sources:

Country Source Source code Granularity Statistics
World Data collected by Johns Hopkins from WHO reports WRD-JHWHO Mostly country-wise, sometimes includes states, provinces or oversea territories confirmed, recovered, dead
Italy Data collected by Davide Magno from Protezione Civile ITA-DMPC Italian regions tested, confirmed, hospitalised, in ICU, quarantined, recovered, dead
United States Data collected by The New York Times USA-NYT Counties confirmed, dead

As part of COVID19DB data base we are bulding three types of tables:

  • EPIDEMIOLOGY (see below): epidemiological data (confirmed, dead, recovered, hospitalised, etc.)
  • MOBILITY (see below): mobility data (movement trends over time by geography, e.g. workplace, parks)
  • GOVERNMENT_RESPONSE (see below): government response to combat COVID-19 (school closures, quarantines, etc.)
  • COUNTRY_STATS (in progress): demographics and heathcare statistics including population, median age, number of hospitals, etc.
  • ADMINISTRATIVE_DIVISION (see below): ISO 3166-1 alpha-2 and alpha_2 country codes, administrative levels 1-3 and their codes, geometry(Multipolygon, 4326), etc

Schema for EPIDEMIOLOGY table and CSV files

Name Data type Short Description
SOURCE VARCHAR Data source
DATE DATE Day of the statistics
COUNTRY VARCHAR English name for the country
COUNTRYCODE VARCHAR(3) ISO 3166-1 alpha-3 country codes
ADM_AREA_1 VARCHAR First-level administrative country subdivision
ADM_AREA_2 VARCHAR Second-level administrative country subdivision
ADM_AREA_3 VARCHAR Third-level administrative country subdivision
TESTED INT Number of people tested
CONFIRMED INT Number of confirmed cases
DEAD INT Number of deaths
RECOVERED INT Number of confirmed who recovered
HOSPITALISED INT Number of confirmed who are/have been hospitalised
HOSPITALISED_ICU INT Number of confirmed who are/have been in the intensive care
QUARANTINED INT Number of confirmed with home quarantine

Note 1: We collect the data as they come from the indicated data sources. It is possible that different sources provide slighlty different statistics.

Note 2: The database is being built and it is likely that the datastructure will change. Check this webpage regularly to be notified about changes.

HOW CAN I CONNECT TO THE DATABASE?

  • host='covid19db.org'
  • port=5432
  • dbname='covid19'
  • user='covid19'
  • password='covid19'
You can also use, for example, the following Python code and save the data into a DataFrame (df): 

         import psycopg2
         import pandas as pd

         conn = psycopg2.connect(
         host='covid19db.org',
         port=5432,
         dbname='covid19',
         user='covid19',
         password='covid19'
         )
         cur = conn.cursor()


          # load EPIDEMIOLOGY table
          cur.execute("SELECT * FROM epidemiology LIMIT 100;")
          df_epidemiology = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

          # load MOBILITY table
          cur.execute("SELECT * FROM mobility LIMIT 100;")
          df_mobility = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

          # load GOVERNMENT_RESPONSE table
          cur.execute("SELECT * FROM government_response LIMIT 100;")
          df_govres = pd.DataFrame(cur.fetchall(), columns=[desc[0] for desc in cur.description])

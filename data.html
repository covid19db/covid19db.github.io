INFECTIONS table

Currently, the schema includes only one table, infections, with the following fields:

  • SOURCE: code describing the source of information
  • DATE: day of the statistics
  • COUNTRY: English name for the country
  • COUNTRYCODE: ISO alpha-3 code of the country.
  • ADM_AREA_1: First-level administrative country subdivision
  • ADM_AREA_2: Second-level administrative country subdivision
  • ADM_AREA_3: Third-level administrative country subdivision
  • TESTED: (cumulative) number of people tested.
  • INFECTED: (cumulative) number of confirmed infected.
  • RECOVERED: (cumulative) number of infected who have recovered.
  • HOSPITALISED: (cumulative) number of infected who have been hospitalised.
  • HOSPITALISED_ICU: (cumulative) number of infected in the intensive care.
  • HOMEQUARANTINED: (cumulative) number of infected with home quarantine.

Note 1: We collect the data as they come from the data sources, we do not check for their consinsentcy nor correctness. It is possible that different sources provide slighlty different statistics.

Note 2: The database is being built and it is likely that the datastructure will change. Check this webpage regularly to be notified about changes.

As part of COVID19DB data base we are bulding the following types of tables:

  • EPIDEMIOLOGY: epidemiological data (confirmed, dead, recovered, hospitalised, etc.)
  • MOBILITY: mobility data (movement trends over time by geography, e.g. workplace, parks)
  • GOVERNMENT_RESPONSE: government response to combat COVID-19 (school closures, quarantines, etc.)
  • ADMINISTRATIVE_DIVISION: ISO 3166-1 alpha-2 and alpha_2 country codes, administrative levels 1-3 and their codes, geometry(Multipolygon, 4326), etc
  • COUNTRY_STATISTICS: demographics and heathcare statistics including population, median age, number of hospitals, etc.
  • WEATHER: weather data

Schema for EPIDEMIOLOGY table and CSV files

Name Data type Short Description
SOURCE VARCHAR Data source
DATE DATE Day of the statistics
COUNTRY VARCHAR English name for the country
COUNTRYCODE VARCHAR(3) ISO 3166-1 alpha-3 country codes
ADM_AREA_1 VARCHAR First-level administrative country subdivision
ADM_AREA_2 VARCHAR Second-level administrative country subdivision
ADM_AREA_3 VARCHAR Third-level administrative country subdivision
TESTED INT Number of people tested [cumulative]
CONFIRMED INT Number of confirmed cases [cumulative]
DEAD INT Number of deaths [cumulative]
RECOVERED INT Number of confirmed who recovered [cumulative]
HOSPITALISED INT Number of confirmed who are/have been hospitalised [cumulative]
HOSPITALISED_ICU INT Number of confirmed who are/have been in the intensive care [cumulative]
QUARANTINED INT Number of confirmed with home quarantine [cumulative]

Note 1: We collect the data as they come from the indicated data sources. It is possible that different sources provide slighlty different statistics.

Note 2: The database is being built and it is likely that the datastructure will change. Check this webpage regularly to be notified about changes.

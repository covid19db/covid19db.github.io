Schema for WEATHER table

We aggregated the daily weather data from Met Office Informatics LAB (for more details see Data Sources) at the second GADM level for every country in the World. For each Level-2 GID region all the points in the grid that are contained in the region have been located. Then for a fixed Level-2 GID region and given variable (e.g. maximum daily precipitation, minimum daily temperature etc.) the average and standard deviation of the considered variable in those grid points is reported.

DATE DATE Day of the statistics
GID VARCHAR Unique geographical ID, for more details see GADM
COUNTRY VARCHAR English name for the country
